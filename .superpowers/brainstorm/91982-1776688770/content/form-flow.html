How should the form flow work?

Pick the approach that feels right for farmers using this app

A

Multi-step Wizard

Step 1: Farmer info → Step 2: Select crop → Step 3: Select pest (filtered) → Step 4: Delivery → Result. Progress bar at top. Each step on its own screen.

① Info → ② Crop → ③ Pest → ④ Delivery
Feels premium. More taps but very guided.
B

Single Page — Dynamic Filter

Same one-page form. User picks crop first — pest dropdown instantly filters to only relevant pests. Fast, minimal clicks. Add pest images as tooltips.

Fastest path. Keeps current feel, just smarter. No page reloads.
C

Two Pages — Crop First

Landing page: pick crop (with big icon cards). Then original form loads with only relevant pests pre-filtered. Visual and guided without full wizard complexity.

⭐ Recommended — visual impact + simplicity
